# Drag & Drop Image Converter
# Конвертер изображений с перетаскиванием

Simple image converter for Windows built with Python.
Простой конвертер изображений для Windows, написанный на Python.

Convert images through a console menu by dragging a file onto a shortcut.
Позволяет конвертировать изображения через консольное меню, если перетащить файл на ярлык.

## Features
## Возможности

- Drag and drop support
- Поддержка перетаскивания файлов

- Convert to PNG, WEBP, ICO, JPG, BMP, and TIFF
- Конвертация в PNG, WEBP, ICO, JPG, BMP и TIFF

- Converted files are saved to the `converted` folder
- Готовые файлы сохраняются в папку `converted`

- Simple console interface
- Простой консольный интерфейс

## Требования
